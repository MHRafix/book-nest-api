## BookNest API

## Technology Used:

- Nest.JS
- TypeScript
- MongoDB
- Mongoose
- Mongoose Aggregation
- JWT - Json Web Token
- Passport
- Web Socket

## Instructions to run the app

- pnpm/npm/yarn install
- pnpm/yarn dev || npm run dev
